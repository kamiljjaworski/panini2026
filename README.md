# Panini FIFA 2026 – Wymiana Kart

Aplikacja Streamlit do dopasowywania wymian kart Panini FIFA 2026.

## Uruchomienie

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
streamlit run app.py
```

Aplikacja otworzy się w przeglądarce pod adresem http://localhost:8501.

> Uwaga: dane są przechowywane w pamięci sesji (demo) i znikają po restarcie.
